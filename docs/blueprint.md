# **App Name**: Satori AI

## Core Features:

- Skill Selection & Self-Assessment: Users can select specific skills (e.g., Python, SQL) and self-rate their proficiency level.
- AI-Powered Task Generation: Dynamically generates unique, micro-task assessments using a large language model, tailored to the selected skill and difficulty.
- Advanced Solution Evaluation: Evaluates submitted solutions not just for correctness, but also for logic flow, error patterns, time behavior, and problem-solving approach using AI models.
- Authenticity Detection: Detects copy-pasted or AI-generated responses within submissions to ensure genuine candidate output.
- Candidate Skill Dashboard: Provides candidates with a personalized dashboard showcasing their skill scores, performance summaries, and progress over time.
- Recruiter Candidate Review: Enables recruiters to view detailed candidate profiles, including skill assessments, authenticity scores, and other relevant metrics to aid in decision-making.
- Skill Data Storage: Securely stores user data, assessment results, skill profiles, and behavioral embeddings using PostgreSQL and a vector database.

## Style Guidelines:

- Primary color: Sophisticated dark blue-purple (#4D4D73) for an intelligent and trustworthy aesthetic in a light scheme.
- Background color: A very light, subtle grey-blue (#F2F2F6) to provide a clean and expansive canvas.
- Accent color: A vibrant cyan-blue (#2AC5E5) for interactive elements and key highlights, conveying a modern and digital feel.
- Headline and body font: 'Inter' (sans-serif) for its modern, clear, and objective appearance, suitable for complex information.
- Code font: 'Source Code Pro' (monospace sans-serif) for displaying code snippets with optimal readability.
- Utilize minimalist, line-based icons throughout the platform, ensuring clarity and reinforcing a sophisticated, digital identity. Consistent system icons should be used for common actions and navigation.
- Implement clean, spacious, and grid-based layouts to present data and assessment flows in an organized and digestible manner. Emphasis on clear hierarchy to guide user attention on dashboards and assessment interfaces.
- Incorporate subtle and functional micro-animations for transitions, feedback on user actions (e.g., submission, completion), and data updates on dashboards, enhancing perceived responsiveness and user experience.
